# EDA_Netflix_Task
EDA and preprocessing on netflix dataset

Project Overview
This project performs data cleaning and exploratory data analysis on Netflix dataset.

Steps Performed
- Data Cleaning
- Handling missing values
- Feature engineering
- EDA visualization

Insights
- Movies are more than TV Shows
- USA produces most content
- Content increased after 2015

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
